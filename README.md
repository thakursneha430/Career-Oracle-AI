# Career-Oracle-AI
## 🔮 Career Oracle AI
✨ Not astrology… but feels like it 👀

## 🌟 About The Project
🔮 Career Oracle AI is an intelligent prediction system that suggests your future career based on your interests, skills, and personality traits.<br>
Built using Machine Learning, this project helps confused souls (yes, us 😭) figure out what they might become in the future 🚀

## 💡 Think of it as:
«🧠 AI + 🎯 Career Guidance + 😏 A little unexpected chaos»

## ⚡ Features
- 🧠 AI-based career prediction
- 🎯 Takes user input (skills, interests, personality)
- 📊 Generates smart predictions
- 🌈 Beginner-friendly ML project
- 💻 Can be extended into a web app (Flask)
